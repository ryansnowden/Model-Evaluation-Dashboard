# Tech Stack - VAD Model Evaluation Dashboard

## Core Application
- **Language:** TypeScript
- **Framework:** React 19
- **Build Tool / Bundler:** Vite

## Styling & UI
- **CSS Framework:** Tailwind CSS (v4)
- **Icons:** Lucide React

## Data Visualization & Motion
- **Charting Library:** Recharts
- **Animations:** Motion (formerly Framer Motion)

## Data Layer
- **Backend/Database:** None (Client-side application relying on static/mock JSON evaluation datasets)
